# Continuous Deployment with Azure and Infrastructure as Code (.NET)

Mise en place d’un pipeline de déploiement continu (CD) avec Azure DevOps pour une application web .NET, en s’appuyant sur l’Infrastructure as Code pour le provisionnement des ressources Azure.

## Fonctionnalités
- Création d’artefacts applicatifs avec pipeline CI
- Provisionnement de ressources Azure via Infrastructure as Code
- Déploiement continu sur plusieurs environnements
- Gestion des variables de configuration et des paramètres applicatifs
- Déploiement de l’application web et des ressources associées

## Technologies
- Azure DevOps
- Microsoft Azure
- YAML / Release Pipelines
- ARM Templates
- .NET 8
- Azure SQL Database
- Azure App Service

## Objectif
Automatiser le provisionnement d’infrastructure et le déploiement applicatif dans un flux DevOps cohérent.
